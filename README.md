# Feux de forêts en Amazonie – Dataviz

Ce dépôt contient :
- `dataviz.py` : script Python qui génère des graphiques (matplotlib) à partir de `amazon.csv`
- `streamlit_app.py` : application Streamlit interactive pour explorer les feux de forêts
- `amazon.csv` : jeu de données (feux de forêts au Brésil)
- `figs/` : dossier où se sauvegardent les figures créées par `dataviz.py`

## Utilisation rapide

```bash
pip install -r requirements.txt
python dataviz.py
